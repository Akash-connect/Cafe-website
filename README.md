# Cafe Website (Django)

A dynamic cafe website built using Django.

## Features
- Menu management via Django Admin
- Contact form (stores messages in DB)
- Category-based menu filtering
- Image upload support

## Tech Stack
- Python
- Django
- HTML/CSS
- SQLite

## Run Locally
```bash
git clone <repo>
cd cafe-website
pip install -r requirements.txt
python manage.py runserver
